# Codex Repository Instructions

## File Modification Policy

- Codex MUST modify only files in the folder **Cult of Vintage**.
- Codex MUST NOT create new files anywhere outside of the folder **Cult of Vintage**.
- Codex MUST NOT modify any other files. outside of the folder folder **Cult of Vintage**
- Cross-file references should be explanatory comments only.

## Change Style

- Prefer minimal diffs.
- Do not refactor unrelated code.
- Preserve existing formatting unless required.

## Vintage Story Mod Environment

- Requires .NET 8 SDK.
- Local dev environment uses the `VINTAGE_STORY` user environment variable pointing to the Vintage Story install directory.
- Do not hardcode absolute paths; if a path is needed, use `VINTAGE_STORY` (or document the expected value in comments).
